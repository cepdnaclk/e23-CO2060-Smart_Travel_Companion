## 📖 About the Project

The **Smart Travel Companion** is a full-stack application developed to enhance the travel experience for tourists visiting Sri Lanka.

Planning a trip often requires using multiple platforms for navigation, accommodation, and sightseeing. This system solves that problem by integrating all essential travel services into a **single unified platform**.

The application leverages smart algorithms to:
- Generate **personalized travel plans**
- Optimize routes for time and cost efficiency
- Provide transportation and location-based guidance

The goal is to **reduce travel stress, save time, and improve overall trip experience** for both first-time and returning visitors.

## ✨ Key Features

### 🗺️ Trip Planning
- Personalized itinerary generation based on user preferences  
- Budget-based travel suggestions  
- Multi-day trip planning support  

### 🚗 Navigation & Routing
- Optimized travel routes  
- Location-based recommendations  
- Integrated map navigation  

### 🏨 Travel Assistance
- Suggestions for accommodations and attractions  
- Transportation guidance (public & private)

### 👤 User Experience

- Cross-platform mobile compatibility

## 🏗️ Architecture

The system uses a **three-tier architecture**:

- **Frontend (React):** Handles UI and user interaction  
- **Backend (Spring Boot):** Processes requests and implements logic  
- **Database (MySQL):** Stores user data and travel information
